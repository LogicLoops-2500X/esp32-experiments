# ⚡ ESP32 Experiments

> Exploring embedded systems through hands-on projects using ESP32 — focusing on how software interacts with the physical world.

---

## 🧠 Overview

This repository documents my ongoing exploration of **embedded systems and IoT** using the ESP32 microcontroller.

Each project focuses on understanding core concepts such as:
- Digital and analog I/O
- Sensor interfacing
- Real-time control
- Hardware–software interaction

All implementations are built from scratch through experimentation and iteration.

---

## 🛠️ Tech Stack

- ESP32
- Arduino IDE
- Embedded C/C++
- Breadboard prototyping

---

## 📂 Projects

### 🔹 Basic Output Control
- Built-in LED Blink  
- Two LED Blink  
- Three LED PWM Brightness Control  

### 🔹 Input Handling
- Button (Pull-Up Configuration)  
- Button (Pull-Down Configuration)  

### 🔹 Analog & Signal Control
- Potentiometer → LED Brightness  

### 🔹 Sensor-Based Systems

### 🔹 Touch & Interaction
- Touch Sensor → Buzzer
